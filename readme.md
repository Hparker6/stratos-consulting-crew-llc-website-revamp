# Stratos Consulting Crew LLC — Website

Company website for Stratos Consulting Crew LLC, deployed on Netlify.

## File Structure

```
├── index.html          # Homepage
├── css/
│   └── style.css       # Global styles
├── js/
│   └── main.js         # JavaScript entry point
├── images/             # Static image assets
├── netlify.toml        # Netlify deployment config
├── .gitignore
└── readme.md
```

## Local Development

Open `index.html` directly in a browser, or use a local server:

```bash
npx serve .
```

## Deployment

This site deploys automatically to Netlify on push to `main`.

## Domain

Google Workspace domain configured via Netlify DNS settings.
